Bring AI-powered natural language product search to your WooCommerce store.

== Description ==

Searchly AI allows your customers to search your WooCommerce store using natural language, such as “red running shoes under $100.”  
It uses powerful AI to generate product embeddings and deliver fast, relevant search results.

= Features =
* Natural language product search
* Fast AI-powered vector matching
* Seamless integration with WooCommerce
* Simple setup with API key
* Works out-of-the-box after plugin activation

== Frequently Asked Questions ==

= How do I set it up? =

After activating the plugin, simply paste your Searchly AI API key from your dashboard into the plugin settings.

= Is there any configuration required? =

No. Once the plugin is activated and your API key is added, Searchly AI begins processing your product data automatically.
